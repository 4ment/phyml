Requirements
---------------------

 * Java (version = 6)
 * maven (version >= 3)


Importing into Eclipse
-------------------------------

 * File -> Import... -> Maven -> Existing Maven Projects
 * Next
 * Browse to this directory
 * Finish


Building outside of IDE
---------------------------------

    mvn clean install



Running gui
-----------

    java -jar target/phyml-gui-binary.jar
